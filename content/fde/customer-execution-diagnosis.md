# 客户执行不动，先判断这四类卡点

客户没有推进任务时，不要先归因于“不配合”。先判断是不会做、没有规则、无人检查，还是目标没有进入真实优先级。

## 适用场景

- 会议上认可方案，会后没有动作
- 已完成培训，但关键记录仍不完整
- 多个角色参与，却没有人对结果负责
- 管理者要求结果，但不检查过程

## 四类执行卡点

| 卡点 | 典型信号 | 对应动作 |
|---|---|---|
| 能力不足 | 不知道怎么操作或完成任务 | 补上手任务、操作清单和示范 |
| 制度缺失 | 会做，但没有统一要求 | 明确规则、字段、质检和异常处理 |
| 责任缺位 | 有任务，但没有固定检查人 | 明确责任人、检查节奏和升级机制 |
| 优先级不足 | 口头重视，但没有时间和资源 | 重新对齐目标、范围、投入和退出条件 |

## 判断顺序

```text
是否会做？
  -> 不会：先补能力
  -> 会：是否有制度约束？
      -> 没有：补规则和质检
      -> 有：是否有人盯执行？
          -> 没有：补责任人和节奏
          -> 有：是否是优先级不足？
              -> 是：升级关键人，重定目标和资源
```

## 标准动作

1. 不先评价配合度，先收集事实：任务有没有完成、记录有没有留下、责任人有没有检查。
2. 每次只处理一个主卡点，避免同时堆培训、制度、考核和复盘。
3. 如果是优先级不足，把风险讲清楚，推动关键决策人重新确认投入。
4. 输出书面诊断：主卡点、证据、本轮动作、责任人和验收。

## 完成标准

- 能用事实指出一个主卡点，而不是笼统批评
- 有对应动作、责任人和时间
- 客户侧知道“接下来做什么才算执行”

## 风险边界

本方法用于对客服务与陪跑判断，不替代组织人事考核。不同客户阶段的优先级不同，不要把一次诊断当成永久标签。
