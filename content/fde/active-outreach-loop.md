# 主动开发闭环：挖潜、触达、孵化到入库

目标不是“发更多信”，而是：

```text
有效线索 = 潜客精准度 × 触达质量与规模 × 孵化转化
```

## 适用场景

- 有目标市场，但缺潜客名单和触达节奏
- 做了触达，却说不清线索是否有效
- 需要把主动开发价值讲成经营结果，而不是工具功能

## 标准闭环

```text
目标市场与买家画像
  -> 潜客挖掘
  -> 背调补全
  -> 个性化多轮触达
  -> 持续孵化
  -> 高意向人工接管
  -> 有效线索入库并进入转化链路
```

## 先定义目标客户

| 检查项 | 常见问题 | 动作 |
|---|---|---|
| 目标市场 | 市场过泛 | 先锁 1-3 个重点市场 |
| 买家类型 | 进口商/品牌商/终端混杂 | 明确买家类型 |
| 主推产品 | 触达内容与产品不匹配 | 先定主推与场景 |

## 看指标找卡点

| 指标 | 判断 |
|---|---|
| 潜客量 | 是否找到足够目标客户 |
| 触达量 | 是否有足够开发动作 |
| 打开/回复 | 内容相关度与名单质量 |
| 高潜识别 | 背调是否筛出值得跟进的对象 |
| 孵化转化 | 未立即回复的价值客户是否被持续推进 |
| 入库率 | 有效线索是否进入统一客户资产 |

## 标准动作

1. 确认市场、买家、产品
2. 建第一批可控规模潜客池
3. 检查背调能否补齐公司与需求相关度
4. 触达必须个性化，不做纯群发
5. 观察打开、回复、高潜识别与孵化
6. 有效线索入库，标注来源、质量、下一步

## 完成标准

- 能说明闭环卡在哪一环
- 触达与入库可追踪
- 有效线索能交给更高转化率链路继续经营
