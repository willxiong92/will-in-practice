# 经营判断与能力调度：先选动作，再选 AI

不要先问“有什么 AI 工具”，先问“客户卡在哪个经营动作”。

## 七类能力群

| 能力群 | 业务含义 | 常见客户问题 |
|---|---|---|
| 商机洞察 · 战略选品 | 卖什么、卖给谁 | 不知道主推产品和目标市场 |
| 品牌定义 · 表达 | 定位与信任表达 | 有产品但表达弱、信任不足 |
| 内容与流量优化 | 上架效率、自然流量 | 发品慢、内容弱、流量差 |
| 营销与客户开发 | 获客触达 | 缺潜客名单和主动触达 |
| 商机管理 · 客户分层 | 响应与经营 | 有线索但响应慢、跟进乱 |
| 成交转化 | 报价、异议、临门推进 | 报价慢、谈判弱、成交卡 |
| 经营分析 · 风控 | 复盘与风险 | 管理层看不清问题与风险 |

## 与经营公式的映射

| 经营公式 | 优先能力群 | 典型任务 |
|---|---|---|
| 更多有效线索 | 营销开发 / 内容流量 | 主动开发、独立站内容与流量 |
| 更高转化率 | 商机管理 / 成交转化 | 新客转化、商机推进 |
| 更高客单价 | 品牌表达 / 成交转化 | 方案价值、组合销售 |
| 更高毛利率 | 经营分析 / 战略选品 | 报价纪律、订单结构、经营数据 |

## 标准任务优先用“编排”

| 任务类型 | 输入 | 输出 |
|---|---|---|
| 独立站首轮诊断 | 站点、产品、市场、流量询盘 | 主结论 + 3 条优先动作 |
| 主动开发诊断 | 市场、买家、产品、触达数据 | 挖潜/触达/孵化卡点 |
| 新客转化诊断 | 询盘、商机、跟进数据 | 阶段卡点与制度动作 |
| 阶段复盘草稿 | 目标、动作、结果 | 复盘初稿 |

## 使用步骤

1. 先归经营象限
2. 再选能力群
3. 标准任务优先走编排流程
4. 输出缺证据时回补业务页输入
5. 有效 Prompt、模板、案例回流平台化沉淀

## 完成标准

- 能说清“为什么选这组能力”
- 能为一个客户场景补齐输入并得到可执行输出
- 服务后至少沉淀 1 个可复用资产

## 风险边界

能力名称和具体 Agent 会随产品迭代变化。本页保留任务驱动的调度方法，不作为功能清单承诺。
