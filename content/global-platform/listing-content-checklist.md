# 国际站发品内容检查清单：标题、主图、商详与关键词

付费推广放大的是内容质量。内容不清楚时，加预算只会买到更多无效点击。

## 适合谁

- 新品上架、老品优化
- 推广花费高但询盘少
- 需要统一运营与业务发品标准

## 上架前总闸

- [ ] 类目与真实产品属性一致
- [ ] 已过 [合规基础](/practice/compliance-basics) 检查
- [ ] 主推款优先级明确（不是全店平均用力）
- [ ] 有可核验的规格 / 认证 / 包装信息来源

## 标题

- [ ] 包含核心品类词 + 关键规格 / 场景（不过度堆砌）
- [ ] 无禁限售词、无夸大承诺（「最好」「保证成交」等）
- [ ] 与主图、属性字段一致，不前后矛盾
- [ ] 同一系列标题结构统一，方便批量维护

## 主图与图组

- [ ] 主图主体清晰、背景干净、可识别品类
- [ ] 有尺寸 / 细节 / 场景 / 包装中至少两类补充图
- [ ] 图上文字少而准，不遮挡关键结构
- [ ] 水印与品牌元素不干扰识别
- [ ] 图片使用权可说明（自有拍摄或已授权）

## 属性与规格

- [ ] 必填属性完整，与实物一致
- [ ] MOQ、交期、定制范围写清
- [ ] 单位、材质、认证字段不留空「乱填」
- [ ] 变体（颜色 / 规格）区分清楚

## 商详结构（建议顺序）

1. 一句话：给谁、解决什么问题  
2. 关键参数表  
3. 应用场景 / 适配条件  
4. 工艺 / 质检 / 认证要点  
5. 包装与物流相关说明  
6. FAQ 或常见异议  

检查：

- [ ] 前屏能看懂卖点，不必滚到底
- [ ] 无大段无关工厂宣传
- [ ] 承诺与合同 / 样品标准可对齐

## 关键词与搜索意图

- [ ] 主词：买家会搜的品类词
- [ ] 长尾：规格、材质、场景、兼容性
- [ ] 避免与产品无关的热搜硬塞
- [ ] 标题、商详、属性中关键词自然出现，不堆砌

## 发布后 7 日复盘

- [ ] 曝光 / 点击 / 询盘是否异常偏低
- [ ] 主图 CTR 是否明显弱于店内均值
- [ ] 询盘问题是否集中在「规格不清 / 交期 / MOQ」
- [ ] 是否需要改主图或补参数，而不是先加预算

## 完成标准

- 任意同事按清单抽查主推品，能在 3 分钟内说清：卖给谁、核心规格、差异点
- 主推品无合规与信息矛盾项
- 改品有版本记录（改了什么、为何改）

## 风险边界

- 各平台字段名、评分与搜索规则会变，以当期后台为准
- 不编造认证与测试数据；无证据不写
- 本清单是内容质量方法，不承诺流量结果
