# 国际站合规基础：禁限售、目标市场与上架前检查

流量问题可以优化；合规红线一旦触发，账户与广告都会停。上架前先过闸门。

## 适用场景

- 新产品上架
- 新市场投放
- 客户被平台提示合规风险
- 团队需要统一上架前检查口径

## 上架前检查

- [ ] 类目与产品实际属性一致
- [ ] 目标市场禁限售清单已核对（含电池、化妆品、医疗、食品接触等敏感类）
- [ ] 标题、主图、属性与禁限售词未踩线
- [ ] 认证、检测、成分、产地等声明有证据
- [ ] 品牌/专利/图片使用权可说明
- [ ] 物流与包装要求与目标市场匹配

## 投放前检查

- [ ] 落地商品页可打开且信息完整
- [ ] 关键声明在广告素材与商详一致
- [ ] 客服口径与页面承诺一致
- [ ] 异常订单与投诉有处理责任人

## 完成标准

- 任一主推品能出示“目标市场 + 风险点 + 证据位置”
- 新人不靠感觉判断能不能卖

## 风险边界

平台规则与各国法规会变化。本页给检查结构，具体条款以平台当期规则与官方法规为准。
