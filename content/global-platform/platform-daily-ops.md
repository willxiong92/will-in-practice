# 国际站日常运营节奏：回复、分级跟进与主动触达

## 场景与问题

**读者**：国际站运营、外贸业务员、带小团队的主管。

**场景**：询盘进得来却跟丢；团队只忙广告；不知道一天先做什么。

**痛点**：

- 回复慢、断头消息（最后一条永远是买家）  
- 所有客户同一跟进频率，重点客户被淹没  
- 主动触达无上限、无记录，变成骚扰或零动作  

## 核心判断

国际站日操的核心不是「今天发了几条营销」，而是：

```text
响应质量（及时、专业、可跟进）
  → 分级节奏（不同意向不同频率）
    → 有边界的主动触达
      → 可复盘的记录
```

底座不稳时加推广，只会放大漏接与无效对话。

## 方法拆解

### 1. 每日三段（建议顺序）

| 时段 | 动作 | 完成标准 |
|---|---|---|
| 开盘 | 清未回询盘/站内信；标紧急 | 无超 SLA 未处理项 |
| 午间 | 推进重点商机；补样品/报价节点 | 重点列表有下一步 |
| 收盘 | 主动触达额度内完成；写跟进记录 | 记录可被他人接手 |

SLA 由团队自定（例如「工作时段 X 小时内首回」），**不写死平台口号数字**。

### 2. 回复质量底线

- 当天收到的询盘，工作时段内完成首回（团队书面约定）  
- 对话尽量避免长期停在「买家最后发言」  
- 产品、交期、包装、物流问题要有可核对答案或明确「待确认时间」  
- 引导到可存档的沟通渠道时，遵守平台规则与公司合规要求  

### 3. 日操优先级池（≠ 买家价值分层）

日操用 **A–D 处理优先级**（谁先回、谁先推进），与 [买家分层 L0–L4](/practice/buyer-tier-playbook) **不是同一套**：L 层看长期价值；A–D 看**本周要处理的紧急与意向**。团队应写一张映射（例：L3+ 且本周有报价节点 → 日操 A）。

| 日操池 | 定义（示例） | 节奏示例 |
|---|---|---|
| A | 明确采购意向，近期可推进 | 更高频，节点驱动 |
| B | 有意向但决策慢 | 周频价值跟进 |
| C | 长线/项目型 | 双周或月度触达 |
| D | 意向不清 | 低频养熟，不空耗 |

频率是**团队约定**，不是行业定律；重点是「有分级、有记录、可检查」。

### 4. 主动触达：类型 + 三原则

**类型（有上限、有记录即可，不写死平台数字）**

| 类型 | 目的 |
|---|---|
| 访客/浏览触达 | 对进店未询盘者破冰 |
| RFQ / 公开需求报价 | 主动抢匹配需求 |
| 推荐/相似买家触达 | 在规则允许下拓展 |

**原则**

1. **有名单**：来源可追溯  
2. **有边界**：每日/每周上限由团队定  
3. **有话术目标**：破冰 / 补资料 / 约下一步，不是群发广告  

站外社媒见 [B2B 社媒获客](/practice/b2b-social-acquisition-playbook)，与站内日操分开记账。

## 风险边界

**适用**：已有国际站店铺与基础商机流。

**不适用**：替代平台官方运营规范；无产品知识时的硬聊；违反平台沟通规则的导流。

**常见错误**：只盯 RFQ/广告后台；无分级；记录写在私人聊天无法交接。

## 行动工具

### 工具 A：日操检查表

- [ ] 未回询盘清零或有升级说明  
- [ ] A 类客户本周节点已更新  
- [ ] 主动触达未超团队上限且有记录  
- [ ] 至少一条商机有「下一步 + 日期」  
- [ ] 异常（投诉/合规提示）已上报  

### 工具 B：周复盘三问

1. 本周漏接或超时首回几次？原因？  
2. A/B 类转化到下一步的比例如何（用自有定义）？  
3. 下周只改 1 个日操动作是什么？  

### 验收标准

- 任意同事能按记录接手 A 类客户  
- 连续两周日操检查表可勾选  
- 推广复盘能区分「流量问题」与「响应问题」
