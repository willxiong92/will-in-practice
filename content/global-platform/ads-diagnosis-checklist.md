# 国际站推广诊断：预算花了没效果时先查这几层

效果差时先定位「卡在哪一层」，再决定是改内容、改响应，还是改投放结构。

## 适合谁

- 花费上升、有效询盘不升
- 只有点击没有对话
- 想建立周复盘而不是凭感觉开关广告

## 诊断顺序（由底到顶）

```text
1 账户 / 合规是否健康
2 主推内容是否清楚
3 询盘响应是否接得住
4 投放结构与出价是否合理
5 是否有目标与复盘
```

对应总图见 [运营框架](/practice/platform-ops-framework)。

## 1. 账户与合规

- [ ] 账户无限制、类目与经营范围匹配
- [ ] 主推品已过 [合规检查](/practice/compliance-basics)
- [ ] 店铺基础服务指标（回复率、纠纷等）未明显拖后腿
- [ ] 无「先投再补资料」的违规风险品

底座不稳时加预算，通常放大无效流量。

## 2. 主推内容

- [ ] 主推款 ≤ 团队能服务好的数量（宁少勿滥）
- [ ] 标题 / 主图 / 商详通过 [发品清单](/practice/listing-content-checklist)
- [ ] 落地信息与广告指向一致（同一产品、同一卖点）
- [ ] 价格、MOQ、交期不在页面上互相矛盾

## 3. 响应承接

- [ ] 高意向询盘有时限响应标准（团队自定并执行）
- [ ] RFQ / 站内信有人值班，不是「下班堆积」
- [ ] 使用 [询盘分级与背调](/practice/inquiry-rfq-playbook)
- [ ] 有效线索进入跟进表，有下次动作

常见假象：广告「有商机」，实际 24 小时后才回，转化被自己耗光。

## 4. 投放结构

- [ ] 先定义目标：有效对话 / 样品 / 订单意向（选一个主目标）
- [ ] 效果类投放与品牌曝光类投放职责分开看，不混成一个「花了多少」
- [ ] 词与定向与主推品相关，排除明显无关流量
- [ ] 预算按主推品集中，避免平均洒到全店垃圾品
- [ ] 有「观察期」再大改，避免天天推翻结构

## 5. 目标成本与复盘

- [ ] 有可接受的商机 / 有效询盘成本区间（团队共识）
- [ ] 每周看：花费、点击、有效询盘、样品、卡点层级
- [ ] 连续低于目标时：先改内容或响应，再调价
- [ ] 记录每次改动：时间、改了什么、预期、结果

## 快速分流表

| 现象 | 优先查 |
|---|---|
| 有花费、几乎无点击 | 出价、匹配、主图/标题吸引力 |
| 有点击、无询盘 | 商详清晰度、价格/MOQ 预期、着陆一致性 |
| 有询盘、无推进 | 响应速度、分级、跟进节奏 |
| 有效询盘成本失控 | 词与品是否匹配、是否推了弱内容 |
| 只某几日暴跌 | 账户状态、素材拒审、库存/价格页变更 |

## 完成标准

- 能指出当前主要卡在 1–5 的哪一层
- 主推品清单与预算分配一致
- 有一周一次的复盘记录，而不是只截后台曲线

## 风险边界

- 各平台广告产品名称、计费与报表口径会变，以官方后台为准
- 不引用无法复核的「提升 xx%」承诺
- 本页是诊断方法，不构成投放收益保证
