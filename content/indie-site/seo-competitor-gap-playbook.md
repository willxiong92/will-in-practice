# SEO 竞品差距分析：抽样可见性，只留下能开工的动作

## 场景与问题

**读者**：独立站运营、内容/SEO 负责人、FDE。

**场景**：新站摸底；排名/询盘相关搜索变弱；季度内容规划；想做数字公关前先看机会。

**痛点**：

- 导出上千词无法行动  
- 商业竞品与 SEO 竞品混谈  
- 只比工具分数，不看真实结果页  
- 列表漂亮，一个月内开不了工  

## 核心判断

有用的竞品 SEO 分析，产出应是 **约 30 天内能开工的动作**，不是仪表盘合集。

本页方法**不绑定任一 SEO 软件**。你可以用表格 + 搜索结果页人工抽样完成；有工具只是加速，不是前提。

```text
1 定义 SEO 竞品（谁抢同一可见性）
  → 2 抽样真实搜索结果与对方页面
    → 3 整理意图/主题差距（限量）
      → 4 评估引用/外链机会可行性（限量）
        → 5 压成 30 天动作表
```

**SEO 竞品 ≠ 商业竞品。**  
**估计流量 ≠ 财务事实。**

## 方法拆解

### 步骤 1：定义 SEO 竞品（2–4 个）

纳入标准（满足两条以上更稳）：

- 在你的主推产品/场景词上反复出现  
- 目标市场语言/国家重叠  
- 内容形态接近（产品站、方案站、内容站）  
- 买家提到「也在看他们官网」  

排除：纯信息门户、只做对方品牌词的站、与你完全不同的客群。

输出：竞品名单 + 各一句「为何是 SEO 竞品」。

### 步骤 2：抽样真实可见性（人工优先）

选 8–15 个**你方在乎的查询**（按外贸 B2B 意图分层）：

| 意图层 | 例子方向（按行业改写） | 你要看的 SERP 类型 |
|---|---|---|
| 交易/采购 | 产品 + supplier / manufacturer / wholesale | 产品/厂家页是否占位 |
| 方案/场景 | 应用场景 + 产品品类 | 方案文是否可询盘 |
| 对比/选型 | vs / alternative / best for | 对比文是否被占据 |
| 问题/认知 | how to / what is（仅当影响采购） | 是否值得做内容入口 |

对每个查询记录：

- 前几名是谁、页面类型  
- 我方是否出现、第几屏  
- 对手页：更新度、参数/认证/询盘路径是否强  

可选：用任意关键词/外链工具加速收集「对方还覆盖哪些相关查询」，但**每条入选机会必须过人工 SERP 抽检**。

### 步骤 3：意图与主题差距（限量 10–15）

从抽样中归纳「对方有、我弱/无、且与询盘相关」的机会，写成动作种子：

| 字段 | 要求 |
|---|---|
| 机会描述 | 一个查询簇或主题，不是孤词堆 |
| 意图层 | 交易 / 方案 / 对比 / 认知 |
| 建议页类型 | 产品页强化 / 支柱指南 / 场景文 / FAQ |
| 差异化 | 参数、认证、交付、行业案例角度（须真实） |
| 关联 money page | 链向哪个转化页（见 [内链架构](/practice/internal-linking-playbook)） |

**硬上限：10–15 条。** 多了就砍到能执行。

与 [AEO/GEO](/practice/ai-seo-aeo-playbook)：若你关心「可被引用的结构」，用该专页检查内容形态；**不要**把某一工具的「AI 可见性分数」当业务 KPI。

### 步骤 4：引用与外链机会可行性（限量 10–15）

只收集「有路径拿到提及/链接」的方向，例如：

- 行业名录、认证公示、展会名录  
- 可投稿的垂直媒体/协会  
- 可提供的独特资产（原厂参数表、测试方法、选型表）  

可行性清单：

- [ ] 我们 30–60 天内做得出对等或更好资产吗？  
- [ ] 是否依赖竞品私人关系（难复制）？  
- [ ] 多个对手都出现（更像通用机会）？  
- [ ] 是否合规、非黑帽？  

**硬上限：10–15 条**，并标「做 / 缓 / 弃」。

### 步骤 5：压成 30 天动作

合并为最多 **8 条**执行项：

| 类型 | 例 |
|---|---|
| 内容 | 写/改 1–3 个高意图页 |
| 架构 | 给 money page 补入口内链 |
| 信任 | 补参数/认证/FAQ 可核验块 |
| 引用 | 推进 1–2 个可行外链/名录动作 |

## 风险边界

**适用**：有明确产品线与可观察搜索竞品的 B2B 站。

**不适用**：抄袭；买卖链接；把第三方流量估计写入合同 KPI；无询盘目标的纯词量竞赛。

**常见错误**：列表过大；不做 SERP 抽检；把商业死敌但从不抢搜索的公司当 SEO 竞品。

**来源说明**：早期母版含特定 SEO 工具操作序列；公开稿改为**工具无关**的抽样与限量执行法，并补外贸意图分层与可行性判断。工具数据仅作可选加速。

## 行动工具

### 工具 A：工作簿（限量）

```markdown
# SEO 竞品差距 · YYYY-MM-DD
SEO 竞品（2–4）：
抽样查询（8–15）：

## 机会种子（≤15）
| 机会 | 意图层 | 页类型 | 差异化 | money page | 30天？ |
|---|---|---|---|---|---|

## 引用机会（≤15）
| 方向 | 资产需求 | 可行性 | 做/缓/弃 |
|---|---|---|---|

## 30 天只做（≤8）
| # | 动作 | 类型 | Owner | 截止 |
|---|---|---|---|---|
```

### 工具 B：完成自检

- [ ] SEO 竞品定义写清，且 ≠ 仅商业竞品列表  
- [ ] 每个机会都抽检过真实结果页  
- [ ] 两张机会表均未超上限  
- [ ] 30 天动作 ≤ 8 且能开工  
- [ ] 未使用未核验的流量估计作承诺  

### 验收标准

- 能解释「为什么是这几个 SEO 竞品」  
- 至少 3 条动作进入任务系统  
- 复盘时对照完成度，而不是再导出更大词表
