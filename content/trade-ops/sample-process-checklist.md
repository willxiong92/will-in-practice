# 外贸样品全流程检查清单：寄前、在途、测试与转单

样品不是礼品，是验证匹配度与推进信任的一环。寄出只是中段，不是终点。

## 适合谁

- 经常寄样但转化低的业务员
- 样品费用与跟进无人统一口径的团队
- 需要把样品阶段写进 CRM 节点的主管

## 寄前确认

- [ ] 用途清楚：测试 / 展示 / 认证预检 / 展会展示
- [ ] 规格、材质、颜色、配件与客户文字确认一致
- [ ] 可提供的样品类型写清：现货样 / 打样 / 等同大货工艺
- [ ] 样品费、运费、可否抵扣大货的规则已说明
- [ ] 交期与快递方式可兑现（含报关/查验缓冲）
- [ ] 敏感品类（电池、粉末、液体等）物流与合规已核对
- [ ] 寄件信息用公司名义；不对工厂暴露买方直连方式（见供应商风险清单）

## 寄出动作

- [ ] 装箱清单与样品标签可识别（型号、批次、数量）
- [ ] 面单、轨迹链接已发给客户
- [ ] 内部记录：寄出日、单号、费用、负责人、关联询盘
- [ ] 若有测试方法建议，随邮件附简短说明（不替代正式标准）

## 在途节点

| 节点 | 检查 |
|---|---|
| 寄出后 1–3 天 | 轨迹正常；异常主动告知 |
| 预计到达日 | 提醒查收；确认是否方便签收 |
| 签收后 1–2 天 | 询问开箱与初步观感；提供技术答疑入口 |
| 签收后约 1 周 | 测试进度；需要补参数 / 补样吗 |
| 签收后约 2 周 | 是否进入正式报价、改规格或关闭 |

## 测试与反馈

- [ ] 主动问：测什么指标、谁测、何时有结论
- [ ] 收集照片 / 数据 / 不合格描述（事实，不争辩语气）
- [ ] 区分：规格误解 / 运输损伤 / 工艺问题 / 期望偏差
- [ ] 改进或重寄必须先征得客户书面/邮件确认（尤其改规格）
- [ ] 结论进档案：通过 / 有条件通过 / 不通过 + 原因

## 转单或关闭

- [ ] 通过：输出正式报价假设（量、交期、付款、MOQ）
- [ ] 有条件通过：列出必须变更项与是否重样
- [ ] 不通过：礼貌收口，保留可再触达条件（新品、降本、新认证）
- [ ] 样品费结算状态更新
- [ ] CRM 状态从「样品中」迁到「报价 / 谈判 / 沉睡」之一

## 完成标准

- 任意活跃样品单能在 1 分钟内找到：单号、规格确认记录、下次跟进日
- 团队不对「寄出后失联」无动作
- 转单或关闭有明确原因，而不是线索蒸发

## 风险边界

- 样品承诺不得超出大货可复制的工艺与成本
- 检测与认证以书面标准为准，口头「差不多」无效
- 与 [客户跟进与维护](/practice/follow-up-maintenance-playbook) 配合使用
