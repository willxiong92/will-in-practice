# 外贸买家画像模板：五维诊断与动态分级

「购买诉求 ≠ 真实痛点」。建档是为了匹配动作，不是贴静态标签炫耀。

## 五维表（复制填写）

| 维度 | 关键问题 | 你的记录 |
|---|---|---|
| 行业/产品 | 标品/定制/混合？主销场景？ | |
| 获客渠道 | 平台/展会/独立站/社媒各占比粗估？ | |
| 团队结构 | 决策人、业务人数、老板参与度？ | |
| 经营类型 | 工贸一体 / 贸易 / 工厂 / 内贸+外贸？ | |
| 真实痛点 | 他说要什么 vs 你判断卡在哪？ | |

## 诉求翻译（示例）

| 客户原话 | 可能真实问题 |
|---|---|
| 要更多新客户 | 老客跟不动、转化低、线索质量差 |
| 要邮件群发功能 | 不愿电话/社媒触达，缺话术 |
| 要数据看板 | 老板看不见过程，不信任团队 |

## 动态分级（示例口径）

| 级别 | 信号 | 跟进密度（自定） |
|---|---|---|
| 潜在 | 询盘未下单 / 长期未返单 | 低 |
| 普通 | 有成交、偶发返单 | 中 |
| 重点 | 金额或频次达线 | 高 |
| 战略 | 高金额+高协同 | 最高 + 专人 |

分级字段建议进 CRM：级别、下次动作、负责人、证据链接。  
与 [买家分层（平台）](/practice/buyer-tier-playbook)、[跟进维护](/practice/follow-up-maintenance-playbook) 对齐口径。

## 经营类型 → 动作倾向

| 类型 | 更常卡在 | 优先动作 |
|---|---|---|
| 平台依赖 | 询盘多转化低 | 首回五步 + 公海盘活 |
| 内贸转外贸 | 流程不熟 | 流程地图 + 单证检查 |
| 小团队 | 资产在个人微信 | CRM 与交接规范 |
| 工贸一体 | 前后端不同步 | 交期/样品信息流 |

## 完成标准

- [ ] 重点买家五维无空白  
- [ ] 级别有调整记录（升级/降级原因）  
- [ ] 周会能按级别分配时间  

## 风险边界

- 不写歧视性标签；分级仅内部经营使用  
- 背调仅限合法公开与客户自愿信息
