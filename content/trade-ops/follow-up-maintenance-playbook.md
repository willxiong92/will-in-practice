# 外贸客户跟进与维护：价值输出、节奏与分级

跟进的目标不是证明你很勤快，而是让客户每次收到你都觉得「有用、可推进」。

## 适合谁

- 询价后长期无回复的业务员
- 样品寄出后不知道怎么跟的人
- 需要统一团队跟进节奏的销售主管

## 跟进五要点

1. **时效**：询价与关键问题尽快回；拖延会直接削弱专业感
2. **停止过度解读**：客户说忙不一定是敷衍；先给价值，再约动作
3. **每次都带价值**：行业信息、交期变化、参数补全、应用场景，而不是只问「有消息吗」
4. **节奏可控**：常见节奏 3–7 天一次，按客户反馈加密或拉长
5. **换角度，不硬缠**：第 5 次、第 10 次跟进仍可能成交，但方式要变

## SLE 结构（节后 / 节点复工邮件）

| 段 | 含义 | 示例方向 |
|---|---|---|
| S Situation | 共同情境 | 假期结束、采购季、行业展会刚结束 |
| L Link | 与业务自然连接 | 新品、交期窗口、上次报价假设变化 |
| E Expectation | 明确期望动作 | 15 分钟通话、确认规格、反馈报价 |

避免：纯寒暄无行动项，或一上来催「尽快下单」。

## 长期未回复：从推销式改价值输出式

**少做：**

- 反复问「收到邮件了吗」
- 连续丢产品册与报价
- 施压句：「不需要请直接说」

**多做：**

- 目标市场政策 / 物流 / 认证相关提醒（只写你能核验的）
- 规格对照表、测试建议、样品使用注意事项
- 交期或原料窗口变化（与对方采购相关）
- 节日短问候后附一个轻量行动项

## 样品寄出后节奏（示例）

| 节点 | 动作 |
|---|---|
| 寄出当天 / 次日 | 单号与轨迹说明 |
| 预计到达日 | 提醒查收，询问是否方便签收 |
| 到达后 1–2 天 | 测试支持、参数答疑 |
| 到达后约 1 周 | 应用场景 / 对比说明（不编造客户案例） |
| 到达后约 2 周 | 是否需要补充样、改规格或正式报价 |

## 客户分级与精力

| 级别 | 含义 | 跟进频率（参考） | 精力倾向 |
|---|---|---|---|
| A | 已成交或高意向 | 1–3 天 | 维护与扩品 |
| B | 有意向未成交 | 5–7 天 | **主战场** |
| C | 潜在 / 低意向 | 2–4 周 | 保持存在感 |

实践判断：多数团队把时间耗在 C 类闲聊或反复催 A 类，却忽视 B 类推进。

## 完成标准

- [ ] 每条活跃线索有级别与下次动作日期
- [ ] 跟进记录能看出「本次提供了什么价值」
- [ ] 节后 / 关键节点邮件能按 SLE 写出
- [ ] 样品阶段有节点，而不是只问「收到了吗」

## 风险边界

- 频率与内容需符合目标市场与渠道规则，避免垃圾邮件与骚扰
- 不编造客户案例；涉密订单细节不得外传
- 文中节奏为实践参考，不是行业统一标准
