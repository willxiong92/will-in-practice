# 外贸展会执行：展前邀约、现场记录与 48 小时跟进

展会效果多半在开幕前决定。展中是收集信号，展后 48 小时决定线索会不会冷却。

## 适合谁

- 首次参展或团队参展动作不统一
- 展后名片很多、成交很少
- 需要把线下流量接回报价与 CRM 的团队

## 三阶段总览

```text
展前（建议提前 ≥ 2 个月）→ 名单、邀请、话术、物料
展中 → 接待结构、记录分级、现场同步
展后 48 小时 → 个性化感谢 + 下一步动作
```

## 展前

### 邀请函最低要素

- 正式 PDF（公司抬头、展会名、时间地点、**展位号**）
- 展位位置示意（可简图）
- 主打产品 / 本届差异点（不超过 3 条）
- 预约洽谈方式（邮箱 / 表单 / 即时通讯，择一写清）

### 目标名单（示例来源，注意合规）

- 目标市场公开目录、行业协会名录
- 过往询盘与成交客户中的相关买家
- 合规可用来源的进口商 / 采购公开信息
- 按优先级排序：必见 / 应邀 / 路过即可

### 四类现场问题：先演练标准答法

| 类型 | 典型问题 | 准备物 |
|---|---|---|
| 价格 | 「比同行贵」 | 差异化证据：认证、交期、质检、服务边界 |
| 产品 | 「和 XX 有何不同」 | 竞品对比表（事实可核验） |
| 公司 | 「做了多久 / 主市场」 | 公司简介 + 可公开案例类型 |
| 合作 | OEM/ODM、付款 | 条件、MOQ、付款方案边界 |

### 物料清单（精简）

| 类 | 项 |
|---|---|
| 文件 | 名片、一页纸、主推报价假设、目录 |
| 展示 | 主推样品、标签规格清晰 |
| 记录 | 线索表（纸质或平板）、笔 |
| 电子 | 电脑 / 电量、资料备份 |

## 展中

### 接待原则

1. 先问需求与用途，再展开产品
2. 不低头刷手机；路过客主动但不纠缠
3. 每个有效对话留下可跟进信息

### 必记字段

- 公司、联系人、渠道
- 品类 / 规格 / 大致量级
- 当前供应商或市场（若对方愿意说）
- 意向：A / B / C
- 下一步：发资料 / 报价 / 约视频 / 验厂意向

## 展后

### 48 小时内

- 全量感谢邮件 / 消息：点名展位上谈过的点
- 附：展位照片或主推一页纸（无敏感价）
- A 类：直接约下一通电话或视频时间

### 一周内

- 意向客户正式报价或样品方案
- 线索进 CRM / 表格：级别、负责人、下次日期
- 团队复盘：到访量、A/B/C 分布、卡点

## 完成标准

- [ ] 展前名单与邀请已发出，不是只做展位搭建
- [ ] 现场记录完整率可抽查
- [ ] 展后 48 小时内完成第一轮触达
- [ ] 每条 A/B 线索有明确下一步与负责人

## 风险边界

- 现场不要承诺未核算的价格与交期
- 名片与对话记录属商业信息，按公司数据规范保存
- 具体展会规则、物流与搭建以主办方通知为准
