# 外贸客户开发：渠道、背调与触达节奏

客户开发要系统化：选渠道、做背调、按节奏触达、把线索沉淀进可跟进资产。

## 四大渠道

| 渠道 | 优势 | 关键点 |
|---|---|---|
| 自建站 / 内容 | 主动流量、品牌资产 | SEO、落地页、询盘承接 |
| B2B 平台 | 流量大、门槛相对低 | 差异化运营、响应速度 |
| 展会 / 线下 | 信任建立快 | 展前名单、展后 48 小时跟进 |
| 社媒 / 主动触达 | 精准、可控 | 画像、价值输出、不硬推 |

## 触达前背调（最低集）

1. 官网：主营、产品、市场定位
2. 关于页：规模、历史、团队信号
3. 产品页：卖什么、定位高低
4. 社媒 / LinkedIn：关键人与业务动态
5. 公开新闻或行业信息：近期变化
6. 海关或公开采购线索（如有合规来源）
7. 邮箱 / 联系方式验证
8. 初步匹配判断：值不值得投入

没有完成最低背调，不建议群发。

## 系列触达节奏（示例）

| 节点 | 内容重点 |
|---|---|
| 第 1 天 | 提供行业价值，不直接硬推产品 |
| 第 3 天 | 分享相关案例或应用场景 |
| 第 7 天 | 介绍匹配点与可提供的下一步 |
| 第 14 天 | 客户见证、交付能力或样品路径 |
| 第 21 天 | 提出一个具体问题，推动对话 |
| 收尾 | 礼貌结束，保留后续入口 |

原则：

- 每封只聚焦一个价值点
- 标题具体，避免空泛口号
- 质量优先于连接数量

## 完成标准

- 每条线索有来源、背调结论、触达记录和下一步
- 能说明为什么投这个客户，而不是“群发碰运气”
- 有效线索进入统一跟进系统，而不是散落在个人聊天里

## 风险边界

不同平台有连接频率、广告与隐私规则，需遵守当地与平台政策。本页不提供绕过平台限制或灰产获客方法。
