# 外贸客诉与危机应对：四步法、交期延误与质量变更

客诉处理的目标不是赢辩论，而是恢复可执行的信任与订单路径。

## 适合谁

- 业务员首次面对质量/交期投诉
- 需要统一升级口径的销售与跟单
- 希望把危机变成可复盘案例的团队

## 四步法

### 1. 承认与对齐事实

- 第一时间确认「发生了什么」，不先甩锅
- 表达重视，避免过度自责导致无边界承诺
- 核对：订单号、批次、数量、照片/视频、检验标准依据

### 2. 提出可执行方案（建议 24 小时内有初步方案）

- 临时止血方案 + 最终方案（可分两段）
- 必须含：动作、负责人、时间点、验证方式
- 站在客户履约压力一侧写方案，而不是只写工厂内部原因

### 3. 执行并同步

- 按承诺节点更新，不等客户来催
- 新障碍主动暴露 + 备选路径
- 交付物可验证：补货清单、检测报告、新交期表

### 4. 回访与预防

- 关闭后回访满意度
- 写清根因与防再发措施
- 需要时给出合理补偿边界（与商务权限对齐）

## 交期延误

**原则：提前预警优于被动暴露。**

预防：

- 接单时保留可兑现缓冲，不按理想产能锁死
- 关键节点主动更新（备料完成、过半、完工、订舱）

已发生延误：

1. 第一时间通知，并带替代方案（分批、加急、改运、换规格窗口）
2. 原因简短，重点放在新交期与保障措施
3. 若影响客户下游销售，评估合理补偿并走内部审批
4. 延误期提高沟通频率

最忌：客户来催才承认 delay。

## 质量问题

**铁律：任何改规格 / 换料 / 改工艺，必须先获客户确认。**

建议流程：

1. 收证（图、视频、不良描述、抽检比例）
2. 内部分清：偶发 vs 系统性
3. 系统性：评估是否影响同组件其他订单（该通知的通知，不掩盖范围）
4. 提出改进方案，**书面确认后再改**
5. 改进样确认通过后，再放大货

## 索赔谈判

1. **核实真实性**：证据是否充分、是否符合约定标准  
2. **责任切分**：设计/来料/工艺/运输/仓储/使用  
3. **损失量化**：退货、返工、运费、下游罚款（有单据再谈）  
4. **方案组合**：补货、折扣、部分退款、换货、责任险路径  
5. **权限**：超出个人授权必须升级，禁止口头超额承诺  

不建议：全盘否认或无原则全盘接受。

## 完成标准

- [ ] 每起客诉有事实记录、方案时间表、关闭结论
- [ ] 延误有预警记录，而不是只有道歉邮件
- [ ] 质量变更有客户确认痕迹
- [ ] 索赔结果与合同/PI 条款可对照

## 风险边界

- 法律与保险问题及时咨询专业人士；本文是业务操作框架
- 不编造检测结论；无证据不认「全责」或「无责」
- 与 [付款与风险](/practice/payment-risk-checklist) 交叉使用
