# 外贸付款与风险控制检查清单

付款方式不是会计题，而是风险分配题。业务员要能在报价前判断：谁先承担货权与资金风险。

## 适合谁

- 外贸业务员与销售主管
- 需要统一团队风险口径的团队负责人

## 签约前

- [ ] 已确认买方主体、国家/地区与公开可核验信息
- [ ] 付款方式与订单金额、信任程度匹配（如 T/T、L/C 等）
- [ ] 合同/PI 写清币种、金额、比例、时间节点与违约责任
- [ ] 已约定质量标准、验货方式与争议处理路径
- [ ] 对首次合作客户设置更保守的预付款或更短账期

## 生产与出货中

- [ ] 订金/约定节点款到账后再进入关键生产节点
- [ ] 交期、规格变更有书面确认
- [ ] 验货与放行标准可追溯
- [ ] 出运文件（发票、装箱单、提单等）与合同一致
- [ ] 放单/交单条件与付款进度对齐

## 异常处理

- [ ] 延迟付款有升级路径（业务 → 主管 → 停产/停运）
- [ ] 质量争议先固定证据，再谈让步
- [ ] 不在口头承诺中扩大信用额度
- [ ] 重大异常写入复盘，形成团队案例

## 完成标准

- 任一订单能说清付款节点与当前风险承担方
- 团队对“什么情况停产/停运”口径一致

## 风险边界

本清单不是法律意见。信用证条款、目的国管制与制裁名单等需结合当期专业意见核验。
