# 外贸业务流程地图与交接检查表

外贸流程不是背 12 个名词，而是每一步都知道交付物、责任人和风险点。

## 流程地图

| 阶段 | 目标 | 关键交付物 | 常见风险 |
|---|---|---|---|
| 客户开发 | 建立有效线索 | 背调记录、触达记录 | 线索无效、无人跟进 |
| 报价 | 快速给出可解释价格 | 报价单、有效期、条款备注 | 拖延、漏运费/汇率条款 |
| 谈判 | 在价值与条件间达成一致 | 谈判纪要、让步边界 | 无依据乱降价 |
| 签约 | 锁定订单条件 | PI/合同、订金到账 | 只发 PI 未收订金就生产 |
| 生产跟单 | 按质按时交付 | 进度表、异常记录 | 熟人单省略验货 |
| 验货质检 | 确认质量标准 | QC 报告、问题闭环 | 无标准、无抽检 |
| 出运 | 完成装运与通关 | 订舱、装柜、报关资料 | 截单时间失控 |
| 文件与回款 | 交单并收齐余款 | 提单/发票/装箱单、回款记录 | 文件错误导致延误 |

## 出货前最低检查

- [ ] 订金/约定付款已确认
- [ ] 生产与验货结论可追溯
- [ ] PI / CI / PL 信息一致
- [ ] 柜型、体积、重量与装柜计划匹配
- [ ] 截单/截关/开船时间已登记
- [ ] 报关资料预留缓冲时间
- [ ] 余款与交单条件已对齐

## 常用单证速查

| 简称 | 用途 |
|---|---|
| PI | 形式发票，签约与确认用 |
| CI | 商业发票，清关用 |
| PL | 装箱单，清关用 |
| QC | 质检报告，验货用 |

## 术语速查

MOQ、ETD、ETA、FCL、LCL、FOB、CIF、T/T、L/C 等术语，团队应统一口径。关键定义以合同与当期贸易条款为准。

## 完成标准

- 任意订单都能指出当前卡在哪一阶段
- 阶段切换有交付物，而不是口头说“差不多了”
- 新人能按检查表完成出货前自检

## 风险边界

本页是业务协作地图，不是法律意见或报关合规保证。单证、商检、目的国要求必须以当期官方与货代/关务专业意见为准。
