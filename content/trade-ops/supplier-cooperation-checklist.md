# 供应商合作与防绕单检查清单

客户看到的是产品与交付；交付能力很大程度取决于供应商体系是否可控。

## 适合谁

- 贸易型 / 工贸协同的业务与采购
- 担心验厂或寄样后被工厂直连客户的团队
- 需要统一供应商准入口径的主管

## 合作前

- [ ] 明确产品规格、质量标准、可接受不良与验货方式
- [ ] 价格、币种、含税口径、调价触发条件写清
- [ ] 交期、产能上限、节假日影响已确认
- [ ] 关键认证与检测能力与订单需求匹配
- [ ] 至少有 Plan B（同品类备份供应商意向）

## 合同 / 书面约定（按公司法务模板裁剪）

- [ ] 规格与验收标准附件
- [ ] 交期与延误处理
- [ ] 付款与开票
- [ ] **客户归属 / 非绕单 / 价格保护**（在可执行范围内）
- [ ] 保密：客户信息、价格、图纸
- [ ] 违约与争议路径

无法签完善合同时：至少邮件纪要锁定「不得直连买方、报价经我方」。

## 信息隔离（日常）

- [ ] 客户沟通走公司渠道，不把工厂拉进同一会话当默认
- [ ] 验厂 / 参观有陪同，不留单聊换联系方式的空档
- [ ] 报价与 PI 用我方抬头；不主动泄露工厂底价给买方
- [ ] 寄样寄件信息用公司名义（在合规前提下）
- [ ] 文件外发前检查是否暴露工厂直联方式

## 双供方与评估

| 维度 | 看什么 |
|---|---|
| 价格 | 合理性、涨价频率 |
| 交期 | 准时率、波动 |
| 质量 | 不良、客诉、一致性 |
| 配合 | 响应、柔性、透明度 |
| 能力 | 产能、设备、认证 |

- [ ] 主力 A + 备份 B 有意识维护，而不是口头「有备胎」
- [ ] 定期对比 A/B，问题供方降级或淘汰
- [ ] 重大订单不把全部风险压在单一工厂

## 利益绑定（实践判断）

- [ ] 回款与订单节奏让工厂感到「长期合作 > 抢一单」
- [ ] 预测与备料沟通提前，减少对方被逼违约的动机
- [ ] 质量争议用数据与标准，不用情绪对抗

## 完成标准

- 主推品类能指出主力与备份供应商
- 核心客户触达路径不经过工厂私邮
- 供应商问题可在评估表上定位，而不是只有「这次倒霉」

## 风险边界

- 具体合同条款以公司法务与当地法律为准
- 信息隔离不得违反平台/展会正当联络规则与反不正当竞争法律
- 本文是业务风险框架，不构成法律意见
