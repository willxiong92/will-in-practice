# SOHO 外贸启动检查清单：储备、选品与防绕单

SOHO 是一人扛采购、跟单、开发、单证的创业形态，需要更强自律与资源整合——**不是**「公司做不好就出来单干」的默认答案。

## 心态与能力门槛

- [ ] 有行业实操积累（常见建议：多年一线后再独立，而非仅数月）  
- [ ] 能接受无打卡时的自我管理  
- [ ] 理解「六无」也能起步，但必须有差异化切入（小客户、细分品、强服务）  

## 启动前储备

### 供应商

- [ ] 主力品线 ≥3 家可合作工厂/渠道  
- [ ] 关键品 A/B 供方（见 [供应商清单](/practice/supplier-cooperation-checklist)）  
- [ ] 书面约定客户归属与价格保护（能签尽签）  
- [ ] 交期、质量、配合度经过小单验证  

### 客户

- [ ] ≥5 个已建立联系的意向  
- [ ] 尽量有 1–2 个可推进订单的窗口（现金流）  
- [ ] 名单合规：不侵犯前雇主商业秘密  

### 资金

- [ ] ≥6 个月生活费缓冲  
- [ ] 样品、快递、域名/站点、差旅预算  
- [ ] 订金与账期周转预留  

### 工具

- [ ] 企业邮箱与域名  
- [ ] 最小可展示站点或专业主页  
- [ ] 沟通与单证模板（PI/报价/合同框架）  
- [ ] CRM 或结构化表格  

## 选品

- [ ] 避开纯红海、纯价格战品类（除非有真成本或服务壁垒）  
- [ ] 评估：利润空间、竞争、复购、认证门槛、物流复杂度  
- [ ] 优先级意识：**供应链可靠度** 往往先于「先找一堆客户」  

## 防绕单（SOHO 更脆）

- [ ] 信息隔离：工厂不直接掌握买方完整联系链  
- [ ] 合同/邮件纪要绑定归属  
- [ ] 利益共享：稳定单量与回款，使「抢一单」不划算  
- [ ] 寄样与文件抬头走己方主体  

## 完成标准

- 储备四项无致命缺口再宣布「已启动」  
- 首单前完成一次完整样品与报价闭环演练  
- 风险清单家人/合伙人知情（资金）  

## 风险边界

- 不鼓励违法挖角客户与侵犯商业秘密  
- 税务、外汇、出口资质按当地法规办理  
- 本文非创业成功保证
